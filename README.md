# 🚀 PriceRadar

**PriceRadar** is a modern price comparison web application that helps users search for products across multiple e-commerce platforms and compare prices in one place. It provides a clean interface, fast search, and an easy way to find the best deals available online.

🌐 **Live Demo:** https://price-radar-hazel.vercel.app/

---

## ✨ Features

* 🔍 Search products across multiple stores
* 💰 Compare prices from different platforms
* ⚡ Fast and responsive search experience
* 📱 Mobile-friendly and responsive UI
* 🎨 Clean and modern interface
* 🚀 Deployed on Vercel

---

## 🛠 Tech Stack

### Frontend

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/ashfaqzahoor/PriceRadar.git
```

### Navigate to the project

```bash
cd PriceRadar
```

### Install dependencies

```bash
npm install
```

or

```bash
pnpm install
```

### Start the development server

```bash
npm run dev
```

or

```bash
pnpm dev
```

Open your browser and visit:

```
http://localhost:3000
```
